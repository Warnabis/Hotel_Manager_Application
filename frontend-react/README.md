# Frontend React — Hotel Manager

React + TypeScript + Vite приложение для управления отелем.

## Запуск

1. Запустите backend (порт 8081) и PostgreSQL
2. `npm install`
3. `npm run dev` — http://localhost:5173

API проксируется через Vite на `http://localhost:8081/api`.

## Разделы

**Публичная часть (стиль booking.com):**
- Главная с поиском номеров
- Каталог номеров
- Оформление бронирования (гость → номера → платёж)
- Каталог услуг
- Регистрация / вход гостя (по email)
- Личный кабинет: бронирования и услуги

**Админ-панель** (`/admin/login`, пароль: `admin123`):
- CRUD для всех 7 сущностей
- Управление связями M:N и N:1 через multi-select
